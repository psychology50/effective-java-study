**기간**

**발표형식**

issue를 활용하여 질의응답. 질문 내용은 파트 담당자가 답변을 달아둔다.  
  - 처음 보는 용어에 대한 설명 요구
  - 글로만 적혀있는 기술에 대한 사용법 요규
  - 예제 코드에 대한 설명
  - 문단 문맥 이해
  - 코드에 대한 분석 요구 (주로 JDK  구현 방식에 대한 상세 설명)
  - 관련 기술 사용 예제 요구
